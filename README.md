# Palmeiras Website

**A fan-made website dedicated to the Sociedade Esportiva Palmeiras.**

### Overview
This project is a basic web application built using HTML, CSS, and JavaScript. It provides a simple search function that allows users to find information about Palmeiras, such as:

* **Latest news:** Stay updated on the latest happenings at Palmeiras.
* **Club history:** Learn about the rich history of the club.
* **Merchandise:** Explore the official Palmeiras merchandise.

### Features
* **Search functionality:** Users can search for specific keywords to find relevant information.
* **Dynamic content:** Search results are dynamically generated based on the user's query.
* **Responsive design:** The website adapts to different screen sizes for optimal viewing.
* **Clean and modern UI:** A simple and easy-to-use user interface.

### Technologies Used
* **HTML:** Structure of the webpage.
* **CSS:** Styling and layout of the webpage.
* **JavaScript:** Dynamic behavior and interactions.
